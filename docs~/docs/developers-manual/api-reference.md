# API リファレンス

詳細なAPIリファレンスについては、各クラスのソースコードとコメントを参照してください。主要なクラスは以下の場所にあります：

- `Runtime/core/` - コアシステム
- `Runtime/channel/` - チャンネルシステム  
- `Runtime/zone/` - ゾーンシステム
- `Runtime/utilities/` - ユーティリティ
