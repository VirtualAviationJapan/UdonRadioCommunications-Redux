# 導入方法

## 基本パッケージのみ利用する場合

1. [ALCOM](https://vrc-get.anatawa12.com/ja/alcom/) か [VRChat Creator Companion](https://vrchat.com/home/download) をインストールしてください。
2. [VPM リポジトリ](https://vpm.virtualaviation.jp/redirect) を追加してください。
3. VPM 経由で `Udon Radio Communications Redux` をインストールしてください。

## SaccFlightと組み合わせる場合

1. [ALCOM](https://vrc-get.anatawa12.com/ja/alcom/) か [VRChat Creator Companion](https://vrchat.com/home/download) をインストールしてください。
2. [VPM リポジトリ](https://vpm.virtualaviation.jp/redirect) を追加してください。
3. VPM 経由で `Udon Radio Communications Redux Saccflight Addon` をインストールしてください。`Udon Radio Communications Redux` は自動的にインストールされます。
